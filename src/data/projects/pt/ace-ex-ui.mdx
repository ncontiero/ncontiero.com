O AceEx é um design system para a web que visa facilitar o desenvolvimento de interfaces consistentes, acessíveis e responsivas. O sistema compreende uma coleção de componentes reutilizáveis, padronizados e documentados que seguem os princípios de design da marca.

## Sobre

Design System com componentes, tokens, configuração do eslint e documentação [Storybook](https://ace-ex-ui.vercel.app/) para os componentes e tokens.

## Guia Rápido

Aqui estão alguns links úteis para começar com o AceEx UI:

- [Documentação](https://ace-ex-ui.vercel.app/) - Aprenda a usar o AceEx UI e veja os componentes no Storybook.
- [Primeiros passos](https://ace-ex-ui.vercel.app/?path=/docs/introduction-getting-started--page) - Configure um novo aplicativo com o AceEx UI ou adicione-o a um projeto existente.
- [Tema](https://ace-ex-ui.vercel.app/?path=/docs/features-theme-tema--page) - Aprenda sobre nossos fundamentos, como cores, espaçamento e tipografia.

## Pacotes

- **[@ace-ex/react](https://www.npmjs.com/package/@ace-ex/react)** — a biblioteca principal de componentes React
- **[@ace-ex/tokens](https://www.npmjs.com/package/@ace-ex/tokens)** — primitivos visuais como tipografia, cor e espaçamento
- **[@ace-ex/eslint-config](https://www.npmjs.com/package/@ace-ex/eslint-config)** — configurações do ESLint
- **[@ace-ex/ts-config](https://www.npmjs.com/package/@ace-ex/ts-config)** — configurações do TypeScript.
- **[@ace-ex/ace-template](https://github.com/ncontiero/ace-ex-ui/tree/main/packages/ace-template)** — inicialize um novo aplicativo Next.js com o AceEx UI.
